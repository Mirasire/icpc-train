# 插入排序

>The value of life lies not in the length of days, but on the use we make of them.

####  程序思路

> 假设总共有n组数据

- 令   i = 2  ->  n
- 令	 j=i -> 0 
- 比较 j 和 j-1 
    - 1.如果j-1 > j 
        - j开始-1
        - 直到 j-1  > j
        - 交换 j 和 i 的值

## Code

```c
for(int i=2,j; i < n; i++)
{
    for(j=i; i>0; j--)
    {
        if(number[j-1] < j[i]) break;
    }
    int temp = number[j];
    number[j] = number[i];
    number[i] = temp;
}
```
